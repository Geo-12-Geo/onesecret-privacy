OneSecret — Privacy Policy

Last updated: 2026-01-14.

Summary

OneSecret is a mobile utility (password generator/manager) that stores all user data locally on the user's device. This application does not collect user data and does not transmit user data to any third party.

Data collection

OneSecret does not collect personal or usage data. All credentials (logins, passwords, secure notes) are entered and stored locally on the device only.

Data transmission to third parties

OneSecret does not share or transmit any user data to external servers, analytics providers, advertisers, or other third parties.

Local storage and security

  • All data is stored locally in the app’s private storage area on the device.
  • Stored data is protected using strong encryption and Android keystore mechanisms where available.
  • The app uses Android biometric APIs (BiometricPrompt) only to authorize access; biometric data is handled by the device and is never shared with the app or transmitted.

Servers, analytics, and ads

There are no external servers, no analytics SDKs, and no advertising SDKs integrated into the app.

Target audience

OneSecret is intended for adult users. The app is not targeted to children and is intended for users aged 18 and older.

Contact

If you have questions about this privacy policy, contact the developer: oneproductstudio.contact@gmail.com

Note: This policy reflects that OneSecret does not collect or transmit user data. If you add server sync, analytics, or third-party services, update this policy and the Google Play safety form accordingly.

Краткое содержание

OneSecret — мобильная утилита (генератор/менеджер паролей). Все данные пользователя хранятся локально на устройстве. Это приложение не собирает пользовательские данные и не передаёт их третьим лицам.

Сбор данных

OneSecret не собирает личные или служебные данные. Все учётные данные (сайты, логины, пароли, защищённые заметки) вводятся и хранятся только локально на устройстве пользователя.

Передача третьим лицам

OneSecret не передаёт и не расшаривает пользовательские данные внешним серверам, аналитическим сервисам, рекламодателям или третьим сторонам.

Локальное хранение и безопасность

  • Данные хранятся локально в приватном хранилище приложения на устройстве.
  • Хранилище защищено с помощью шифрования и механизмов Android Keystore (где доступно).
  • Биометрическая аутентификация осуществляется через системный API Android (BiometricPrompt). Биометрические данные обрабатываются системой и никогда не передаются приложению и не отправляются.

Серверы, аналитика и реклама

Приложение не использует внешние серверы, аналитические SDK и рекламные SDK.

Целевая аудитория

OneSecret предназначено для взрослых пользователей и не ориентировано на детей. Рекомендуемый возраст пользователей — 18 лет и старше.
